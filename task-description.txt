При вирішенні завдання використовуйте один з наступних фреймворків - Symfony, Laravel.
Бажано використовувати Symfony.

# Завдання. Реалізувати блог з постами розбитими по категоріях.
Категорії, пости повинні створюватися, редагуватися, віддалятися усіма користувачами.

Повинна бути можливість додати коментарі до вже створених категорій і постам.
Коментарі повинні додаватися з допомогою Ajax.

Записувати в базу даних IP і інформацію про браузер користувача для кожної нової сесії.
Відстежувати нових користувачів потрібно на всіх сторінках проекту.
Вивести текст з інформацією скільки користувачів з якого браузера перейшли (наприклад "Chrome: 8, Firefox: 5, ...") на всіх сторінках проекту (наприклад в хедері).

# Категорія Повинна мати поля name description Сторінки: список категорій посад сторінки редагування / створення категорії сторінка категорії (на сторінці повинні бути список посад категорії, коментарі категорії) # Пост Повинен мати поля: name - required content - required file - максимальний розмір 2 Mb

# Сторінки: сторінки редагування / створення поста сторінка поста (на сторінці повинні бути інформація про пост і коментарі поста)

# Коментар Повинен мати поля: author - required, повинно містити два слова, обидва з великої літери content - required created_at - дата створення, заповнюється автоматично

Всю зроблену роботу викласти на свій аккаунт в Github. Побажання Написання тестів буде плюсом. Використання CSS Bootstrap буде плюсом.
Використання npm буде плюсом. Використання Docker буде плюсом.
Якщо для вирішення завдання вибрали Symfony, використовуйте його стандартні бандли і підходи.
